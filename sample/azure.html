Linux 가상 머신

Azure Linux Virtual Machines는 Red Hat, Ubuntu 또는 원하는 Linux 배포를 사용하는 안전하고 가상화된 주문형 대규모 인프라를 제공합니다. 빠른 시작, 자습서 및 샘플을 사용하여 Linux VM을 생성, 구성, 관리, 업데이트 및 크기 조정하는 방법을 알아봅니다.
